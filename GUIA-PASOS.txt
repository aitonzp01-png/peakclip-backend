  GUÍA COMPLETA — PEAKCLIP: PUESTA EN MARCHA
══════════════════════════════════════════════════════════════════

1. EJECUTAR SQL EN SUPABASE DASHBOARD

   Ve a: https://supabase.com/dashboard/project/tjuiourlpbwivjzyewav/sql/new

   Pega y ejecuta el contenido de: peakclip-app/schema.sql
   (incluye todas las tablas, RLS y triggers)

2. CONFIGURAR STRIPE

   a) Crear precios en Stripe:
      - Producto "Creator" → Price ID: price_creator (o el que generes)
      - Producto "Pro" → Price ID: price_pro
      - Actualizar price_id en dashboard/page.js si cambian

   b) Webhook:
      - Endpoint: https://peakclip-backend.onrender.com/stripe-webhook
      - Evento: checkout.session.completed
      - Copiar signing secret → STRIPE_WEBHOOK_SECRET

3. DEPLOY BACKEND EN RENDER

   1. Ir a: https://dashboard.render.com/blueprints
   2. Conectar repo: aitonzp01-png/peakclipnuevo
   3. Render detecta render.yaml automáticamente
   4. Agregar env vars secretas:
      OPENAI_API_KEY
      SUPABASE_URL
      SUPABASE_SERVICE_KEY
      STRIPE_SECRET_KEY
      STRIPE_WEBHOOK_SECRET
   5. Deploy
   6. URL: https://peakclip-backend.onrender.com

4. DEPLOY FRONTEND EN VERCEL

   1. Ir a: https://vercel.com/new
   2. Importar: aitonzp01-png/peakclipnuevo
   3. Root directory: peakclip-app
   4. Agregar env vars:
      NEXT_PUBLIC_SUPABASE_URL
      NEXT_PUBLIC_SUPABASE_ANON_KEY
      SUPABASE_SERVICE_KEY
      NEXT_PUBLIC_BACKEND_URL = https://peakclip-backend.onrender.com
   5. Deploy
   6. URL: https://peakclip-app.vercel.app

5. CONECTAR FRONTEND CON BACKEND

   1. En Vercel: cambiar NEXT_PUBLIC_BACKEND_URL → URL de Render
   2. En Render: FRONTEND_URL → URL de Vercel
   3. Redeploy ambos

6. VERIFICAR

   ✓ GET https://peakclip-backend.onrender.com/ → {"status": "PeakClip API running"}
   ✓ GET https://peakclip-app.vercel.app/ → Landing page carga
   ✓ Login/registro funciona
   ✓ Generar clips desde URL de YouTube
